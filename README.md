# C Language Practice

A collection of C programs written during my B.Tech Computer Science journey at Centurion University. This repo covers core concepts from basics to intermediate-level topics.

## Topics Covered

- **Basics** — variables, data types, operators, control flow
- **Arrays & Strings** — 1D/2D arrays, string functions
- **Functions** — user-defined functions, recursion
- **Pointers** — pointer arithmetic, pointers with arrays/functions
- **Structures** — structs, nested structs, arrays of structs
- **File I/O** — reading and writing files in C
